# Computer Networks Routing Assignment

## Task 1: Basic Configuration
* Configured basic router management parameters including hostname, console passwords, and line configurations.

## Task 2: Static Routing
* Configured manual static routes on all three routers to achieve full network reachability across the subnets.
* Verified connectivity using `show ip route` and `ping` commands.

## Task 3: Dynamic Routing
### Part A: RIPv2
* Removed static paths and implemented RIP version 2 classless dynamic routing across the topology.
### Part B: EIGRP
* Configured EIGRP using Autonomous System (AS) 10 for faster network convergence and dynamic neighbor adjacencies.
